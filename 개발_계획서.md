# ME-IN 플랫폼 단계별 개발 계획서

## 🎯 **개발 목표**
ME-IN 플랫폼을 완전한 브랜드-인플루언서 매칭 플랫폼으로 발전시키기 위한 체계적인 개발 계획

## 📅 **Phase 1: 기본 인프라 및 인증 시스템 강화** (1-2주)

### 1.1 프로필 관리 시스템
- [ ] **사용자 프로필 페이지** (`/profile`)
  - 개인정보 수정
  - 프로필 이미지 업로드
  - 소셜 미디어 링크 관리
  - 언어 및 시간대 설정

- [ ] **브랜드 전용 프로필**
  - 회사 정보
  - 브랜드 카테고리
  - 타겟 지역/언어
  - 예산 범위

- [ ] **인플루언서 전용 프로필**
  - 팔로워 수
  - 주요 플랫폼 (Instagram, TikTok, YouTube)
  - 콘텐츠 카테고리
  - 협업 이력

### 1.2 데이터베이스 스키마 확장
- [ ] **Campaigns 테이블**
- [ ] **Applications 테이블** (인플루언서 신청)
- [ ] **Messages 테이블** (채팅)
- [ ] **Notifications 테이블**

### 1.3 보안 및 권한 관리
- [ ] **Row Level Security (RLS) 정책**
- [ ] **역할 기반 접근 제어**
- [ ] **API 엔드포인트 보안**

---

## 📅 **Phase 2: 캠페인 관리 시스템** (2-3주)

### 2.1 브랜드 캠페인 생성
- [ ] **캠페인 등록 폼** (`/campaigns/create`)
  - 기본 정보 (제목, 설명, 카테고리)
  - 이미지/비디오 업로드
  - 예산 및 기간 설정
  - 타겟 인플루언서 조건

### 2.2 캠페인 관리 대시보드
- [ ] **브랜드 대시보드 개선**
  - 진행 중인 캠페인 목록
  - 신청자 관리
  - 성과 지표

### 2.3 인플루언서 캠페인 탐색
- [ ] **고급 검색 및 필터링**
  - 카테고리별 필터
  - 예산 범위 필터
  - 지역별 필터
  - 팔로워 수 기준

---

## 📅 **Phase 3: 매칭 및 신청 시스템** (2-3주)

### 3.1 AI 매칭 알고리즘
- [ ] **자동 매칭 시스템**
  - 브랜드 요구사항 분석
  - 인플루언서 프로필 매칭
  - 점수 기반 순위 시스템

### 3.2 신청 및 승인 프로세스
- [ ] **인플루언서 신청 시스템**
  - 캠페인 신청 폼
  - 제안서 작성
  - 포트폴리오 첨부

- [ ] **브랜드 승인 시스템**
  - 신청자 검토
  - 승인/거절 처리
  - 피드백 제공
